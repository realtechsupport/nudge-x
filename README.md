## Nudge

<i> Nudge junkyard AI models to do new things.</i>
<br><br>

<p align="center">
  <img src="https://github.com/realtechsupport/nudge-x/blob/main/Rotating-Earth.png?raw=true" alt="Mining sitesin project Nudge"/>
</p>

This repository contains all the code, documents and documentation required for the NUDGE project, an attempt to make good use of old AI models. In the scrapyard, nothing is ever truly gone, it is just waiting to be melted into something else. Or as others have observed, new media do not erase old media, they assign new positions for them.
<br><br>
If you want to rebuild the text generation pipeline, follow the 
[installation instructions.](https://github.com/realtechsupport/nudge-x/blob/main/install_instructions.txt).
<br>
If you want to recreate the UI, follow the instructions in the
[application deployment.](https://github.com/realtechsupport/nudge-x/blob/main/frontend/notes/earth_deployment_instructions.md)
